How to update:

Mac: Open Terminal

Mac 1: Type cd to reset the cd

Mac 2: Type git clone https://github.com/HttpAnimations/FYC to copy the new files

Mac 3: You might need to delete the old FYC Folder to do that in Terminal Do next step

Mac 4: rm FYC

Linux: Open Terminal

Linux 1: Type cd /home/USER to reset the cd

Linux 2: Type rm /home/USER/FYC

Linux 2: Type git clone https://github.com/HttpAnimations/FYC

Windows: There is no way to do update in Windows yet so you need to redownload
